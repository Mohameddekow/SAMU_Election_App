## Techniology Used in this project.
## Language
  * kotlin
## Architecture 
  * MVVM
  
## Dependency injection
  * Dagger_hilt

    * Firebase Authentication
* ## Images Storege

* ## Data Storage
    * Fiarebase Firestore

  * liveData
